# kap-archive

To install dependencies:

```bash
bun install
```

Development dependencies that wont be installed:

- sass (Ruby Sass 3.7.4)
- nodemon

To develop:

```bash
bun dev
```

To run:

```bash
bun run index.jsx
```

This project was created using bun. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
